Repositorio de ejercicios, prácticas y pequeños proyectos realizados durante el curso de **Diseño y Desarrollo Web con HTML5, CSS3 y JavaScript** de la UNED.

---

## 📚 Sobre el curso

Este curso introduce el desarrollo web moderno mediante el uso integrado de HTML5, CSS3, JavaScript y jQuery. Se trabajan tanto los fundamentos de estructuración y diseño web como la creación de páginas dinámicas e interactivas.

---

## 🛠️ Contenido del repositorio

| Área | Temas |
|------|-------|
| **HTML5** | Ejercicios prácticos de estructura y semántica |
| **CSS3** | Maquetación y estilos |
| **JavaScript** | Prácticas, expresiones regulares, manipulación del DOM |
| **jQuery** | Eventos, interactividad e introducción a la librería |
| **Proyectos** | Mini proyectos frontend |
